Travelodge London Covent Garden - Digital Redesign Concept

A modern, high-performance landing page concept designed for the Travelodge London Central Covent Garden hotel. This project focuses on elevating the user experience through immersive visuals, smooth motion interactions, and a "zero-clutter" booking interface.

Note: This project serves as a technical proof-of-concept to demonstrate modern frontend capabilities for the hospitality sector.
